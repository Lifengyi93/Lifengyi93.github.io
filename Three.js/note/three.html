# 更丰富的场景

前面的场景太过单调，这一小节，我们将对几何体添加材质，灯光和阴影，构建更加丰富和逼真的场景效果。在上一节例子的基础上，我们添加如下代码。

# 添加平面

  //平面
  let planeGeometry = new THREE.PlaneGeometry(200,300,30,30);
  let planeMaterial = new THREE.MeshBasicMaterial({color:0xcccccc});
  let plane = new THREE.Mesh(planeGeometry,planeMaterial);
  plane.rotation.x = -0.5*Math.PI;
  scene.add(plane);

# 修改立方体

上一节创建了一个简单的立方体，我们这里做一些调整,修改了这个立方体的材质，修改后的代码如下：

  //几何体
  let cubeGeometry = new THREE.BoxGeometry(40,40,40);
  let cubeMaterial = new THREE.MeshBasicMaterial({color:0xff0000});
  let cube = new THREE.Mesh(cubeGeometry,cubeMaterial);
  cube.position.set(0,20,60);
  scene.add(cube);

# 添加球体

 //球体
 let sphereGeometry = new THREE.SphereGeometry(30,30,30);
 let sphereMaterial = new THREE.MeshBasicMaterial({color:0x7777ff});
 let sphere = new THREE.Mesh(sphereGeometry,sphereMaterial);
 sphere.position.set(30,40,-30);
 scene.add(sphere);

# 添加辅助轴

  //辅助轴,数值定义了这个辅助轴的长度
  let axes = new THREE.AxisHelper(60);
  scene.add(axes);

# 添加网格

  //网格
  let grid = new THREE.GridHelper( 300, 30, 0x000000, 0x000000 );
  grid.material.opacity = 0.4;  //设置网格透明度
  grid.material.transparent = true;
  scene.add( grid );
