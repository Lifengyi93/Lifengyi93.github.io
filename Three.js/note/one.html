# 起步

Three.js（当前版本为 r109）提供了几种源码获取方式，在这里我介绍几种比较简单的方式。

# CDN 引入

使用 BootCDN 提供的免费 CDN 加速服务（同时支持 HTTP 和 HTTPS 协议），引入的版本可以查看这里，直接引入方式为：





 








<!DOCTYPE html>
<html lang="en">
<head>
  <title>Title</title>
  <script src="https://cdn.bootcss.com/three.js/109/three.js"></script>
</head>
<body>
  <script>
    // 这里书写Three.js相关代码
  </script>
</body>
</html>

# 获取源码

Three.js的源码可以在这里获取,选择“Clone or download”存置于本地，关于获取github上的代码保存至本地的知识这里不做详细介绍，git相关 知识请移步廖雪峰git教程。 获取的源码解压缩之后结构如下：

  • build：里面含有 Three.js 构建出来的 JavaScript 文件，可以直接引入使用，并有压缩版；
  • docs：Three.js 的官方文档；
  • editor：Three.js 的一个网页版的模型编辑器；
  • examples：Three.js 的官方案例；
  • src：这里面放置的全是编译 Three.js 的库文件，后续教程中学习导入外部模型到浏览器中显示会用到其中的一些库；
  • 其他：开发环境搭建、开发所需要的文件，如果不对 Three.js 进行二次开发，用不到。

在build文件夹下有three.js、three.min.js这样个文件，前者用于开发的完整版，后者是压缩过的，是用于在互联网上部署的版本，本地开发时使用前者，在这里将three.js拷出保存至创建的项目中，再在head标签中引入即可:





 








<!DOCTYPE html>
<html lang="en">
<head>
  <title>Title</title>
  <script src="pathName/three.js"></script>
</head>
<body>
  <script>
    // 这里书写Three.js相关代码
  </script>
</body>
</html>

这里的pathName指的是项目中存放three.js对应的路径

# 在Vue项目中引入Three.js

直接通过cnpm引入

npm install three --save

安装完成后，在页面中引入three.js并使用，在所调用的vue页面中引入的代码为：

import * as Three from 'three';

vue 项目中与非vue项目的构造函数这里产生了变化，主要是用Three替换了THREE，后面教程会附相关代码，这里只需了解这一概念即可

//非vue项目中
let scene = new THREE.Scene();
//vue项目中
let scene = new Three.Scene();
