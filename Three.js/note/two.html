# 基本的3D场景搭建

在这里我们需要知道在浏览器中展示3D模型后文简称为(web 3D),需要哪些要素。

  1. 场景(Scene):相当于一个容器；模型，灯光等都需要添加到场景中；
  2. 相机(Camera):是一双帮助我们观察3d世界的眼睛；
  3. 渲染器(Renderer)：场景渲染输出的目标;
  4. 渲染(render):执行渲染操作;

# 创建要素并渲染展示

先看一个综合示例，这里将一个3D 矩形模型渲染到浏览器中:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>three.js</title>
    <link rel="stylesheet" href="../main.css">
    <script src="../three.js"></script>
</head>
<body>
    <script>
        // 场景
        let scene = new  THREE.Scene();
        
        // 相机
        let camera = new THREE.PerspectiveCamera(50,window.innerWidth/window.innerHeight,1,1000);
        camera.position.set(200,200,200);
        camera.lookAt(scene.position);
        
        // 渲染器
        let renderer = new THREE.WebGLRenderer();
        renderer.setSize(window.innerWidth,window.innerHeight);
        document.body.appendChild(renderer.domElement);

        //几何体
        let cubeGeometry = new THREE.BoxGeometry(100,100,100);
        let cubeMaterial = new THREE.MeshNormalMaterial();
        let cube = new THREE.Mesh(cubeGeometry,cubeMaterial);
        scene.add(cube);

         // 灯光
        let light = new THREE.PointLight(0xffffff);
        light.position.set(-30,160,60);
        scene.add(light);
        
        // 渲染
        renderer.render(scene,camera);
    </script>
</body>
</html>
