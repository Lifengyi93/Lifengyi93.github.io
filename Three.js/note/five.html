# 给场景添加交互

three.js提供了多种场景控制库，其中的OrbitControls可以很好的对整个场景进行旋转，平移，缩放。

# OrbitControls

首先引入插件，文件地址在官方案例的examples/js/controls/OrbitControls.js,将这个js文件拷贝至项目中存放并在页面的head标签里引入

<script src="/pathName/OrbitControls.js"></script>

之后再创建OrbitControls对象控制场景并执行渲染，代码如下:

//场景控制
let controls = new  THREE.OrbitControls(camera);
controls.addEventListener('change',render)

//渲染        
function render(){
  renderer.render(scene,camera);
}
render();

现在就可以通过鼠标操纵场景进行平移，旋转，缩放了。

# 控制优化

如果浏览three.js中的一些官方示例，你会发现这些示例中的控效果比默认的效果操作起来舒服很多，这是因为OrbitControls有很多参数来控制显示效果，比如缩放阻尼，自旋转，平移速度等。 其中的参数控制如下:

 let controls = new  THREE.OrbitControls(camera);

//鼠标控制是否可用
  controls.enabled = true;
 
//聚焦坐标
  controls.target = new THREE.Vector3();
 
//最大最小相机移动距离(景深相机)
  controls.minDistance = 0;
  controls.maxDistance = Infinity;
 
//最大最小鼠标缩放大小（正交相机）
  controls.minZoom = 0;
  controls.maxZoom = Infinity;
 
//最大仰视角和俯视角
  controls.minPolarAngle = 0; // radians
  controls.maxPolarAngle = Math.PI; // radians
 
//水平方向视角限制
  controls.minAzimuthAngle = - Infinity; // radians
  controls.maxAzimuthAngle = Infinity; // radians
 
//惯性滑动，滑动大小默认0.25
  controls.enableDamping = false;
  controls.dampingFactor = 0.25;
 
//滚轮是否可控制zoom，zoom速度默认1
  controls.enableZoom = true;
  controls.zoomSpeed = 1.0;
 
//是否可旋转，旋转速度
  controls.enableRotate = true;
  controls.rotateSpeed = 1.0;
 
//是否可平移，默认移动速度为7px
  controls.enablePan = true;
  controls.keyPanSpeed = 7.0;	// pixels moved per arrow key push
 
//是否自动旋转，自动旋转速度。默认每秒30圈
  controls.autoRotate = false;
  controls.autoRotateSpeed = 2.0; // 30 seconds per round when fps is 60
 
//是否能使用键盘
  controls.enableKeys = true;
 
//默认键盘控制上下左右的键
  controls.keys = { LEFT: 37, UP: 38, RIGHT: 39, BOTTOM: 40 };
 
//鼠标点击按钮
  controls.mouseButtons = { ORBIT: THREE.MOUSE.LEFT, ZOOM: THREE.MOUSE.MIDDLE, PAN: THREE.MOUSE.RIGHT };
