# 添加阴影

不同的材质对光源的反映并不相同，我们在前一节的示例中在给几何体和平面赋予材质时调用的MeshBasicMaterial()方法添加的是基础材质， 这种材质不会对场景中的光源产生反应，而只会以指定的颜色渲染物体，所以我们不得不改变cube,sphere,plane的材质。

# MeshLambertMaterial

MeshLambertMaterial是一种可以对光照产生反应的特殊材质，我们将上一节创建的几何体材质由MeshBasicMaterial修改为MeshLambertMaterial。修改后的代码如下高亮行所示：

three.js里有两种材质可以对光源产生反应：MeshLambertMaterial和MeshPhongMaterial



 






 






 




  //几何体
  let cubeGeometry = new THREE.BoxGeometry(40,40,40);
  let cubeMaterial = new THREE.MeshLambertMaterial({color:0xff0000});
  let cube = new THREE.Mesh(cubeGeometry,cubeMaterial);
  cube.position.set(0,20,60);
  scene.add(cube);

  //平面
  let planeGeometry = new THREE.PlaneGeometry(200,300,30,30);
  let planeMaterial = new THREE.MeshLambertMaterial({color:0xcccccc});
  let plane = new THREE.Mesh(planeGeometry,planeMaterial);
  plane.rotation.x = -0.5*Math.PI;
  scene.add(plane);

  //球体
  let sphereGeometry = new THREE.SphereGeometry(30,30,30);
  let sphereMaterial = new THREE.MeshLambertMaterial({color:0x7777ff});
  let sphere = new THREE.Mesh(sphereGeometry,sphereMaterial);
  sphere.position.set(30,40,-30);
  scene.add(sphere);

在浏览器中运行效果如下图所示

其实到我们快要完成了，球体和立方体看起来好多了，只是没有添加阴影，看起来有点别扭，所以这仍然不是我们想要的效果。

# 添加阴影

渲染阴影需要消耗大量的计算资源，因此，在默认情况下three.js并不产生阴影，但是生成阴影还是很简单的。修改几处代码即可：

  1. 渲染器一定要允许产生阴影；
//开启阴影
renderer.shadowMapEnabled = true;
  1. 灯光要开启投射阴影；
//开启灯光投射阴影
light.castShadow = true;
  1. 物体要开启投射阴影；
//cube,sphere开启投射阴影
cube.castShadow = true;
sphere.castShadow = true;
  1. 物体要接收投影；
//cube,sphere,plane开启接收阴影
cube.receiveShadow = true;
sphere.receiveShadow = true;
plane.receiveShadow = true;

设置renderer.shadowMap.type = Three.PCFSoftShadowMap；可以增加抗锯齿效果，使阴影更真实。

# AmbientLight

从上图可以看到，物体的暗部细节消失，整个场景看起来很暗，怎么调整能使场景效果更好呢？ 我们给场景添加一个AmbientLight(环境光)，并修改下之前添加到场景中的灯光参数：

//添加环境光
let ambientLight = new THREE.AmbientLight(0xcccccc,0.7)
scene.add(ambientLight);

将之前的

let light = new THREE.PointLight(0xffffff);

修改为

let light = new THREE.PointLight(0xffffff,0.5);
